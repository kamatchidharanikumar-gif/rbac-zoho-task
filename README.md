# Config-Driven RBAC System (Zoho Task)

## Overview
This project demonstrates a config-driven Role Based Access Control (RBAC) system.
Admin and User roles are supported, with table-level and field-level permissions
stored in database and enforced by the backend.

## Tech Stack
- Backend: Node.js + Express
- Database: SQLite
- Frontend: HTML, CSS, JavaScript

## Features
- Username & password authentication
- Admin and User separation
- Config-driven permissions (not hardcoded)
- Field-level access control
- Backend-enforced authorization
- Simple and clean UI
